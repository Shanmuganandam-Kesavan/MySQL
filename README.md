# Library Book Management System (MySQL Project)

This project demonstrates how to manage library book records and member borrow history using SQL.

## Features
- Member registration
- Book inventory management
- Book issue and return tracking

## How to Run
1. Open MySQL Workbench or phpMyAdmin.
2. Run `library_management.sql` to create the database and tables.
3. Modify and query the tables as needed.
